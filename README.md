# Ejecución utilizando una Terminal de Linux

Copiar .env.prod to .env

```sh
cp .env.prod .env
```

Llenar el archivo .env con las credenciales de estudiante regular

Ejecutar el siguiente script en la terminal

```sh
chmod +x app.sh
./app.sh
```
